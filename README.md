# Krish Patel — Portfolio

Personal portfolio site inspired by [advaithmalka.github.io](https://github.com/advaithmalka/advaithmalka.github.io), built with Next.js and Tailwind CSS.

## Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Deploy (GitHub Pages)

```bash
npm run deploy
```
